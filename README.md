# Tienda de Mascotas

_Es una pagina de ecommerce para comprar animales_



### Pre-requisitos 📋

_Se Necesita react-router-dom la cual provee la navegabilidad al sitio. Puede ser instalado de la siguiente forma_

```
npm install react-router-dom
```
